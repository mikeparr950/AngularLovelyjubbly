Examples of some of the websites that I have developed are listed below, along with a brief description of each website. The majority of the websites below can be visited on the internet simply by clicking on their image, although most of the websites do require a User Name/Password in order to gain access to them. Therefore I have also provided screenshots of every website.


Felix Matklubb is a loyalty program website for Felix customers in Sweden. Customers can enter codes found on product packaging to accumulate points in order to redeem rewards or coupons. Points can also be gained in other ways such as inviting friends. The site also provides information on various products and is mobile optimised and features Facebook integration. The main technologies used on this project were ASP.NET MVC, C#, SQL Server, HTML, CSS, Bootstrap, LINQ 2 SQL and Javascript.

Carbon Desktop is used across a wide range of companies worldwide including Arla Foods, Dairy Crest and Northern Foods. The site takes in data feeds from a variety of sources both manual and automated, and is used in the monitoring and targeting of a companies’ utility usage via various custom reports and graphs, enabling them to reduce energy usage and thereby cut utility costs across the company. It also allows for the management of all associated documentation and reports enable the company to monitor data against local legal requirements, for example in the area of carbon reduction. The main technologies used on this project were ASP.NET, C#, LINQ 2 SQL, JQuery, Web Services, Javascript, and SQL Server.

the Gameplan NFLC League website is a website for the NFLC league of the popular Play-By-Mail game Gameplan. It is updated fortnightly and features the latest results, league standings, statistics, team pages, a forum, league history and links to the rules of the game. The main technologies used on this project were ASP.NET Core MVC, C#, Angular, Entity Framework, SQL Azure, Typescript, Web API, Bootstrap, SASS and Angular Material.

Ihmeveikko is a website that brings together various services related to the sale of property, such as selling the property, moving, loans and cleaning. The site is used by both property sellers and people trying to sell their services such as estate agents, cleaning companies, loan companies and moving firms. Property sellers can list their property on the site and view information about providers of various services and receive quotes for these services. The site also has a community aspect in that users can rate service providers and provide feedback, so other users can make choices based upon the feedback service providers have received. The main technologies used on this project were ASP.NET Core MVC, C#, Web API, SQL Server and Entity Framework.

this is a web application that enables staff within LBM to manage usage of Datavision 2, their online business data service. Datavision 2 allows customers to query LBM's marketing database and build datasets that are relevant to their requirements, which they can then save and download upon payment. DV2 Admin allows LBM staff to manage users, their permissions, payment rates, payment methods and other user information. It also has a reporting section which includes reports on users and their usage of Datavision, orders and revenue, and also allows users to view and print invoices. DV2 Admin also gives administrators access to all saved datasets and allows them to enable or disable whether or not the customer can download them, based on whether payment has been made or not. The main technologies used on this project were ASP.NET, C#, SQL Server and SQL Server Reporting Services.

Xerox LMS provides administrators, managers and users within Xerox with a CRM system with which to manage their leads, diary events, contacts and all other related activities. The website is used to manage and track leads through the complete process, from making a new contact, to assigning a lead, to arranging an appointment, to making the sale, and the system is also able to hold competitor sales data which can be used to target potential future sales opportunities. A number of reports can also be run against this data. The main technologies used on this project were ASP.NET, C#, SQL Server, XSLT, SQL Server Reporting Services and Flash.

Chat 2 enables SIP users to make and receive telephone calls to and from the telephone network (PSTN) irrespective of their geographical location. SIPCall not only offers call charge free IP to IP communication but also the ability to make and receive telephone calls. The Chat 2 website allows customers to open SIPCall accounts online via credit card payment, secured by SSL. Existing users can also use the site to securely credit their account, view their call logs and transaction history, change their PIN, and view international inbound codes for calling card access. The main technologies used on this project were ASP.NET, C# and SQL Server.
